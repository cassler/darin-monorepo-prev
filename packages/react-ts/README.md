# Workflower Core - React

## Quick Start

1.) Clone the repository
2.) From root of project run: `$ yarn workspace @wf/react-ts run build`;
3.) Now start a lite-server with `$ yarn workspace @wf/react-ts light-server -s . -p 7000`
