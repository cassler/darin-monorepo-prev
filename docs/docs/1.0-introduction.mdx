Previous releases can be found [here](https://ghe.coxautoinc.com/Darin-Cassler/workflower-monorepo/releases)

### Quick Install (Non-Developer)

1. Unzip the contents to your Desktop (or anywhere convenient on your computer).
2. Open the newly created folder.
3. Double click `index.html` to open Workflower in your default browser.
